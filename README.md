\# AURA AI



\## Transform Customer Data into Business Decisions



AURA AI is an AI Growth Operating System for modern businesses. It helps companies understand customer behavior, predict churn, recommend products, forecast sales, analyze customer reviews, and generate AI-powered business decisions.



\## Project Status



Sprint 1 completed: Product planning, SRS, product vision, roadmap, and system architecture.



\## Core Features



\- Business Health Score

\- AI Decision Center

\- Action Center

\- Customer Churn Prediction

\- Customer Segmentation

\- Product Recommendation System

\- Sales Forecasting

\- Review Sentiment Analysis

\- Explainable AI Insights

\- Executive Dashboard



\## Tech Stack Planned



Frontend: React.js, Tailwind CSS  

Backend: FastAPI  

Database: PostgreSQL / Supabase  

Machine Learning: Python, Pandas, Scikit-learn  

Deployment: Vercel, Render, Supabase  

Version Control: Git \& GitHub
